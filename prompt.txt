You will write a minimal map projection viewer into @main.c. Nothing fancy.
Only WRITE into this file, without looking at anywhere else.
You will not read anything. only write.

Use raylib 6.0, no other external libs.
This program will take in an equirectangular image from argv[1].

The window will be resizable, with default size 800x600.
On top of the window, show buttons: [ Equirectangular ] [ Mercator ] [ Sphere ]
Clicking on the buttons will change the projection (replace the central map/sphere object)

Recall The Mercator projection:

x = R * lon
y = R * log( tan(pi/4 + lat/2) )

have it scale to the width of the window by default

Flat maps (Equirectangular, Mercator) will be drawn as a cube.

Sphere will be drawn as sphere.

Use shaders for projection.

Controls:
- Left click: rotate central map/sphere object around center
- Shift left click: pan
- Scroll wheel: zoom
