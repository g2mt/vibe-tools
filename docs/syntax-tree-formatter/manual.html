Syntax Tree Formatter Manual

Overview

This tool helps format and manipulate Lisp-style syntax trees (e.g., [S [NP John] [VP [V eats] [NP apple]]]).

Buttons

Format

Pretty-prints the entire tree with indentation and newlines. Flat nodes (nodes with no nested brackets) are kept on a single line.

Unformat

Collapses the entire tree into a single line, removing all unnecessary whitespace.

Select Block

Selects the smallest bracketed expression [...] that contains the cursor.

BeforeAfter
[S [NP John|] [VP eats]]
 
[S [NP John] [VP eats]]

Wrap Block

Wraps the current block in a new parent node labeled "X". The cursor is placed at the label for easy renaming.

BeforeAfter
[S [NP John] [VP eats]]
 
[S [X| [NP John]] [VP eats]]

Unwrap Block

Removes the brackets of the current block, effectively promoting its contents to the parent level.

BeforeAfter
[S [NP John] [VP eats]]
 
[S John| [VP eats]]

Indent (») and Dedent («)

Shifts the selected lines (or the current line) by 2 spaces. Mimics Vim's > and < behavior.

BeforeAfter
[S
  [NP John]
  [VP eats]]
 
[S
    [NP John]
  [VP eats]]

Copy unformatted

Collapses the tree to a single line and copies the result to your clipboard without changing the text in the editor.
